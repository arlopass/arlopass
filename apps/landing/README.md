# @arlopass/landing

Astro-based marketing and documentation site for Arlopass.

## Purpose

- Product marketing pages
- Developer docs
- Security and architecture guides
- Search indexing assets (`llms.txt`, Pagefind)

## Development

```bash
pnpm --filter @arlopass/landing dev
```

## Build

```bash
pnpm --filter @arlopass/landing build
```

The build step generates static assets and search index files.

## Preview

```bash
pnpm --filter @arlopass/landing preview
```
