# @byom-ai/adapter-tooling

Development utilities for building, testing, and validating BYOM AI provider adapters.

> [!NOTE]
> This package is a placeholder at v0.1.0. Planned tooling includes a contract test harness, adapter scaffolder, and local sandbox runner.

## Planned Features

- `create-byom-adapter` — Scaffold a new adapter project
- Contract conformance test suite — Validate adapter implementations
- Local sandbox runner — Test adapters in an isolated environment
- Manifest linting — Validate adapter manifests against the schema
